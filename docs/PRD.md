## 1. 项目概述

### 1.1 目标
构建一款类 PyCharm 布局的桌面编辑器，深度集成 Claude Code CLI，解决以下 CLI 痛点：
- 无法直接粘贴/拖拽图片到对话
- 终端与编辑器分离，上下文切换频繁
- 缺乏可视化文件管理和代码编辑体验

### 1.2 目标用户
- 已使用 Claude Code CLI 的开发者
- 需要 AI 辅助编程但偏好 GUI 的用户

### 1.3 核心价值主张
**"一个窗口，完成编码 + AI 对话 + 终端操作"**

## 2. 功能需求

### 2.1 优先级分级

#### P0（必须完成 - MVP）

| 编号 | 功能模块 | 需求描述 | 验收标准 |
|------|----------|----------|----------|
| P0-01 | 核心布局系统 | 左侧边栏（文件树）、中央编辑器区、底部终端区，可拖拽调整尺寸 | 三栏布局稳定运行，拖拽分隔线调整宽度/高度，最小尺寸限制生效 |
| P0-02 | 文件管理 | 文件树展示、新建/删除/重命名文件和文件夹 | 支持右键菜单操作，操作后文件树实时刷新，错误时弹窗提示 |
| P0-03 | 代码编辑器 | 多标签页、语法高亮、自动保存 | 支持至少 5 个标签页，高亮覆盖 JS/TS/Python/Java/HTML/CSS，自动保存间隔 ≤ 3 秒 |
| P0-04 | Claude 对话集成 | 流式输出、思考模式（收起/展开） | 流式响应延迟 ≤ 500ms，思考模式可切换，历史记录保留 |
| P0-05 | 图片交互 | 拖拽/粘贴图片到对话、预览、发送 | 支持 PNG/JPG/GIF/SVG，单张 ≤ 5MB，粘贴后显示缩略图预览 |
| P0-06 | 终端集成 | 底部终端，完全替代系统终端 | 支持常见 shell 命令（bash/zsh/powershell），与系统终端行为一致 |
| P0-07 | CLI 进程管理 | 管理 Claude Code CLI 进程，通信协议 | 进程可启动/停止/重启，通信无丢包，异常时自动重连 |

#### P1（重要 - 迭代优化）

| 编号 | 功能模块 | 需求描述 | 验收标准 |
|------|----------|----------|----------|
| P1-01 | 搜索功能 | 全局文件搜索、代码内搜索替换 | 支持正则，搜索结果高亮，替换支持预览 |
| P1-02 | 主题切换 | 深色/浅色主题、自定义主题 | 切换无闪烁，主题色值可配置 |
| P1-03 | 快捷键系统 | 自定义快捷键绑定 | 支持常见 IDE 快捷键（VSCode/PyCharm 预设） |
| P1-04 | 多工作区 | 支持打开多个项目文件夹 | 切换工作区时状态保留，内存占用可控 |
| P1-05 | 设置面板 | 编辑器设置、AI 参数设置、终端设置 | 设置持久化，修改即时生效 |

#### P2（增强 - 长期规划）

| 编号 | 功能模块 | 需求描述 | 验收标准 |
|------|----------|----------|----------|
| P2-01 | 插件系统 | 支持第三方插件扩展 | 提供插件 API，至少 3 个示例插件 |
| P2-02 | 协作功能 | 多人实时协作编辑 | 基于 CRDT，延迟 ≤ 200ms |
| P2-03 | 版本控制 | Git 集成，可视化 diff | 支持提交、分支、冲突解决 |
| P2-04 | 远程开发 | SSH 远程项目开发 | 连接稳定，文件同步延迟 ≤ 1s |

## 3. 非功能需求

### 3.1 性能指标
- 冷启动时间 ≤ 5 秒
- 文件树加载 1000 个文件 ≤ 1 秒
- 编辑器打开 10MB 文件不卡顿
- 内存占用 ≤ 500MB（空闲状态）

### 3.2 兼容性
- 支持 Windows 10+/macOS 12+/Linux Ubuntu 20.04+
- 最低分辨率 1366×768

### 3.3 安全
- 不存储用户 API Key（使用系统 Keychain）
- 敏感操作需确认（删除文件、执行命令）

## 4. 界面原型（文字描述）

```
+----------------------------------------------------------+
| 菜单栏 | 工具栏                                              |
+----------+-----------------------------------------------+
|          |                                               |
|  文件树   |              编辑器区域（多标签）              |
|  (侧边栏) |                                               |
|          |                                               |
|          |                                               |
+----------+-----------------------------------------------+
|                      底部终端                             |
+----------------------------------------------------------+
```

- 侧边栏宽度默认 250px，可折叠
- 底部终端高度默认 200px，可收起/展开
- 中央编辑器支持水平/垂直分屏

## 5. 验收标准汇总

### 5.1 MVP 验收（P0 全部完成）
1. 安装包可正常安装并启动
2. 可打开本地项目文件夹，文件树正确显示
3. 可在编辑器中编辑代码，自动保存生效
4. 可与 Claude 进行对话，支持粘贴图片
5. 底部终端可执行命令，与系统终端行为一致
6. 连续运行 8 小时无崩溃

### 5.2 上线标准
- 通过 supervisor 审查
- 通过 owner 验收
- 无 P0 级别 Bug
- P1 功能完成率 ≥ 60%

## 6. 风险与依赖

| 风险 | 影响 | 缓解措施 |
