# Claude Code 桌面编辑器 — 全量测试用例文档

> 版本：v1.0  
> 日期：2026-06-18  
> 测试负责人：QA 测试工程师  
> 技术栈：Electron + React + Monaco Editor + XTerm + Zustand

## 1. 测试概述

### 1.1 测试目标
验证 Claude Code 桌面编辑器在功能、兼容性、性能三个维度的质量，确保发布前无阻断级缺陷。

### 1.2 测试范围
| 维度 | 覆盖模块 |
|------|----------|
| 功能测试 | 文件管理、代码编辑、Claude 对话、图片交互、菜单功能、终端集成 |
| 兼容性测试 | Windows 10/11、分辨率（1080p/2K/4K）、DPI 缩放（100%/125%/150%） |
| 性能测试 | 启动时间、大文件打开、24 小时稳定性 |

### 1.3 测试环境
- OS：Windows 10 Pro (19045) / Windows 11
- Node.js：v22.22.3
- Electron：latest stable
- 显示器：1080p (100% 缩放) / 2K (125% 缩放) / 4K (150% 缩放)
