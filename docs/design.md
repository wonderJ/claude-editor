## 2. 交互流程

### 2.1 用户操作路径：打开项目 → 编辑文件 → 对话交互 → 终端命令

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  启动应用  │ → │ 打开项目  │ → │ 浏览文件  │ → │ 编辑文件  │ → │ 自动保存  │
│          │    │ (Ctrl+O) │    │ 文件树点击 │    │ 标签页打开 │    │ (≤3s)   │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
                                                                    │
                                                                    ▼
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  终端命令  │ ← │ 执行命令  │ ← │ 查看结果  │ ← │ 发送消息  │ ← │ 对话交互  │
│  (Ctrl+`)│    │ 回车执行  │    │ 终端输出  │    │ Ctrl+Enter│    │ 粘贴图片  │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
```

### 2.2 详细交互流程

#### 路径 A：打开项目
1. 用户点击 `File → Open Folder` 或按 `Ctrl+O`
2. 系统文件选择器打开
3. 用户选择项目文件夹
4. 文件树加载显示（≤1s 1000文件）
5. 最近打开记录更新

#### 路径 B：编辑文件
1. 用户在文件树点击文件
2. 编辑器区域打开新标签页（或切换到已有标签）
3.  Monaco Editor 加载，语法高亮生效
4. 用户编辑代码
5. 自动保存触发（防抖 ≤3s）
6. 标签页标题显示保存状态（● 未保存 → 已保存）

#### 路径 C：对话交互
1. 用户点击侧边栏 🤖 图标或 `AI → New Chat`
2. 对话面板展开（右侧，默认 350px）
3. 用户在输入框输入问题或粘贴图片
4. 点击发送或按 `Ctrl+Enter`
5. Claude 流式响应（首字延迟 ≤500ms）
6. 思考模式自动收起，可点击展开
7. 历史记录保留，可上下滚动查看

#### 路径 D：终端命令
1. 用户点击底部终端标签栏或 `Ctrl+``
2. 终端展开（默认 200px）
3. 用户输入命令，按 `Enter` 执行
4. 输出实时显示
5. 终端与系统终端行为一致

### 2.3 拖拽交互

```
文件树拖拽 → 编辑器区域
  └─ 打开文件到新标签

图片拖拽 → 对话面板输入框
  └─ 显示缩略图预览，点击发送

分隔线拖拽 → 调整面板尺寸
  └─ 侧边栏：最小 200px，最大 400px
  └─ 终端：最小 150px，最大 50% 窗口高度
  └─ 对话面板：最小 300px，最大 500px
```

- 遮罩：背景 `rgba(0,0,0,0.6)`

### 4.8 提示/通知

#### Toast 通知
- 背景：`--bg-secondary`
- 边框：左侧 3px 颜色（success/warning/error/info）
- 圆角：`--radius-md`
- 阴影：`--shadow-md`
- 内边距：`--space-md`
- 位置：右上角，间距 `--space-lg`
- 自动消失：3s（成功）/ 5s（警告/错误）

#### 确认弹窗
- 类型：删除文件、执行危险命令
- 标题：明确操作
- 内容：影响说明
- 按钮：取消（次级）+ 确认（危险色 `--error`）
